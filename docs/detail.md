# RestoX Function  Specification
สรุปฟังก์ชั่นทั้งหมดขอระบบ RestoX

## 1. Platform
    - จัดการร้าน สร้าง, ลบ, แก้ไข
    - ดูรายชื่อร้านทั้งหมด
    - ดูสถิติภาพรวมทุกสาขา (ค่อยระบุเจาะจง)
    
## 2. Store or manager 
    - จัดการพนักงาน สร้าง ลบ แก้ไข
    - โต๊ะ เพิ่ม ลบ แก้ไข
    - จัดการสถานะโต๊ะ
    - อนุมัติ สั่งซื้อวัตถุดิบ
    - อนุมัติ การจัดการรายการอาหาร
    - ดูรายการวัตถุดิบ
    - ดูยอดขาย รายวัน เดือน ปี 
    - ดูสถานะการจอง

## 3 kitchen
    - เปลี่ยนสถานะ order
    - แจ้งจัดการ รายการอาหาร เพิ่ม ลบ แก้ไข ไปยัง manager 
    - แจ้งคำสั่ง สั่งซื้อวัตถุดิบ ไปยัง manager 
    - จัดการสถานะโต๊ะ
    - แจ้งวัตถุดิบที่ใช้เพิ่มในกรณีพิเศษ (option)
    - รับ order จากโต๊ะ
    - ดูสถานะการจอง
    - รับสัญาณ การคิดเงิน

## 4. Table
    - สั่งอาหาร ยกเลิกหากยังไม่ทำ
    - แจ้งคิดเงิน (รวม, แยก)
    - ดูสถานะ order ดู queqe
    - แจ้งเตือน order ที่เสร็จ
    - เรียกพนักงาน/แจ้งเหตุพิเศษ

## 5. Customer
    - จองโต๊ะ ยกเลิก
    - รีวิว
    - ดูเมนู


## ระบบกลาง 
    - สมัครสมาชิก
    - เข้าสู่ระบบ
    - ออกจากระบบ
    - แก้ไขข้อมูลผู้ใช้

## ระบบเบื้องหลัง  
    - langauge Rust
    - login use jwt 
    - จัดการ session  token 
    - ดู log ผู้ใช้
